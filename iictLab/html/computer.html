Central Processing Unit (CPU)

The CPU is the brain of the computer that performs instructions from programs by performing basic arithmetic, logic, control, and input/output operations specified by the instructions.

Memory (RAM/ROM)

Memory consists of RAM (Random Access Memory) and ROM (Read-Only Memory). RAM temporarily stores data that the CPU is actively using, while ROM stores firmware and essential system information.

Storage (HDD/SSD)

Storage devices like HDD (Hard Disk Drive) and SSD (Solid State Drive) store data persistently. HDDs use spinning disks to store data, while SSDs use flash memory, offering faster access speeds.

Software vs Hardware

Software Hardware
Instructions that tell the computer what to do Physical components of a computer
Not tangible; stored electronically Tangible; can be touched or seen
Examples include operating systems, applications, and games Examples include CPU, memory, and motherboard
Can be easily updated or modified Changes often require physical alterations
Runs on hardware and allows users to perform tasks Enables the execution of software programs
